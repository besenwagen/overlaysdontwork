Misidentified Language

كيف يؤثر الهواء الملوث على صحة أمعائك؟

Explanation

The above passage of text was written in Arabic. It says "How does polluted air affect the health of your bowel" (a story headline from the BBC Arabic site, February 5, 2019). The passage's language has not been identified. As a result, it will not be read properly to screenreader users.

To automatically fix this, an overlay product would need to:

  1. Detect that the passage of text is in a different language than the identified language on its container
  2. Determine which language it is
  3. Mark that passage with the correct ISO language code

These are all possible. In fact, this could even be done in the site visitor's browser. However, doing so requires analyzing every element in the DOM that holds any text in it and attempting to determine its language.
